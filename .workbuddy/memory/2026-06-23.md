## game-sfx-generator Skill 创建

基于 ai-music-design-doc.md 第五大点创建了音效生成 Skill：

### 结构
- `SKILL.md` — 完整英文使用说明（~10KB），含 4 大能力模块描述
- `scripts/sfx_utils.py` — 核心音频合成库：WAV I/O、正弦/三角/方波/噪声生成、FM合成、滤波器（LP/HP/BP）、Schroeder混响、混音、包络
- `scripts/generate_ui_sfx.py` — 3 个空灵UI音效（route_drag_start / route_node_pass / route_complete）
- `scripts/generate_species_sfx.py` — 8 个物种音效（4种FM合成叫声 + 4种噪声合成环境音）
- `scripts/download_free_sfx.py` — 免费商用素材搜索/下载工具（Pixabay/Mixkit/Freesound curated links）
- `references/sound_design_specs.md` — 完整音效设计参数表

### 技术方案
- 动物叫声（水鸟/斑头雁/角马/林蛙）：FM合成，carrier 220-3700Hz，modulator 110-650Hz，index 2.5-5.0
- 环境音（蝴蝶嗡鸣/鲑鱼跃水/鳗鱼滑行/海龟呼吸）：粉噪/棕噪/白噪分层滤波 + 包络塑形
- UI音效：加性合成（sine叠加）+ 滤波噪声 + Schroeder混响
- 素材下载：curated links 到 Pixabay/Mixkit/Freesound CC0 资源

### 依赖
- numpy（必需，用于数组运算和波形生成）
- scipy（可选，增强滤波功能）
- soundfile（可选，替代内置 wave 模块）

### 输出
- 打包文件：`output/game-sfx-generator.zip`（~25KB）
- 下载资源文件：`needownload/SOUND_RESOURCES.md`（免费素材指南）
- Skill 位于：`.workbuddy/skills/game-sfx-generator/`

## 空灵交互音效生成（game-audio-generator skill）

使用 `game-audio-generator` skill 的 `audio_utils.py` 底层库，编写定制脚本生成了两组核心交互音效：

### 生成文件
| 文件 | 时长 | 大小 | 用途 |
|------|------|------|------|
| `output/route_node_pass.wav` | 0.28s | 49KB | 连线经过节点时触发 |
| `output/route_complete.wav` | 0.93s | 164KB | 到达终点释放鼠标时触发 |

### 音效设计
- **route_node_pass**: 木叩点击层 (bell 800Hz) + 3颗水晶铃音层 (1-2.1kHz) + 轻微气流扫频 (1-6kHz)，小房间混响 0.25s
- **route_complete**: 温暖低频呼气 (pad 300Hz) + 上升颂钵滑音 (D4→F4) + D大调五声琶音 (5音) + 5颗水晶粒子散落，中厅混响 0.85s
- 格式：16-bit 44100Hz 立体声 WAV
- 基于 `ai-music-design-doc.md` 5.1b/5.1c 设计规格，调整时长和密度以匹配用户空灵需求
- 生成脚本：`output/generate_sfx_custom.py`
