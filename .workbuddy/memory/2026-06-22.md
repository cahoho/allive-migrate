## 生态迁徙解谜游戏配乐生成

基于 game-audio-generator skill 的 audio_utils 合成引擎，编写自定义 Python 脚本生成了 75 秒无缝循环配乐。

**交付物**: `output/eco_migration_lydian_87bpm_20260622_233233.wav` (13.2 MB, 75.00s)

**规格**:
- Key: D Lydian (D E F# G# A B C#)，87 BPM, 4/4
- 6 轨编排: glass pad / marimba / soft piano / alto flute / warm strings / harp arpeggios
- 4 段情绪弧: 0-20s 马林巴独奏+glass pad → 20-45s 钢琴+长笛进入 → 45-65s 弦乐+竖琴高潮 → 65-75s 回归马林巴独奏
- 混音按用户精确规格: marimba center -3dB, piano left -5dB, harp right -6dB, glass pad wide -10dB, strings center -7dB, flute right -8dB
- 中型厅堂混响 1.8s decay 30% wet
- 3 秒交叉淡入淡出实现无缝循环
- 自定义 marimba_tone (triangle+harmonics+noise) 和 glass_pad_tone (detuned sine pair) 合成器

**技术发现**:
- audio_utils 的 `save_stereo_wav` 用 `packed += struct.pack()` 在循环中拼接字节串，O(n²) 复杂度导致 75 秒音频写入崩溃
- 解决方案: 用 `bytearray` 预分配 + 手动字节写入替代字符串拼接
- 用 `array.array('f')` 替代 Python list 存储 330 万采样，节省 ~80% 内存

**脚本位置**: `output/generate_eco_migration.py`
